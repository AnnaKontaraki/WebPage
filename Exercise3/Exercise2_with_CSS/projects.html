Τμήμα Πληροφορικής - ΟΠΑ

Στη σελίδα αυτή μπορείτε να βρείτε μερικά απο τα projects με τα οποία έχω ασχοληθεί κατα την φοίτησή μου αυτά τα 4 χρόνια.

  • Κατανεμημένα Συστήματα

    Στο μάθημα αυτό, χρησιμοποιώντας το Android Studio και σε γλώσσα Java αναπτύξαμε μια παραλλαγή του γνωστού "Spotify".

  • Αλληλεπίδραση Ανθρώπου-Υπολογιστή

    Κι εδώ πάλι χρησιμοποιώντας το Android Studio και σε γλώσσα Java μας ζητήθηκε να δημιουργήσουμε μια διεπαφή που θα αφορούσε μια οικιακή συσκευή και σκοπός της ήταν να διευκολύνει μια ομάδα ατόμων. Εμείς,λοιπόν, αναπτύξαμε μια εφαρμογή για πλυντήριο ρούχων που θα διευκόλυνε κυρίως τους ηλικιωμένους αλλά και όσους ανθρώπους δυσκολεύονταν με τη χρήση του πλυντηρίου.

  • Προγραμματισμός στην C++

    Στη διάρκεια του μαθήματος μας ζητήθηκε να αναπτύξουμε ένα παιχνίδι στον υπολογιστή χρησιμοποιώντας την γλώσσα C++ και γραφικά. Εγώ, λοιπόν, υλοποίησα το "Arcade Game: Pong (1972 Atari)" με θέμα "Avengers".
